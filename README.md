# How to Start the Program

## Required Dependencies
- Node
- NPM

## Steps to Start the Program
1. Navigate to the project directory:
    ```bash
    cd covid-visualiser
    ```
2. Install the necessary packages:
    ```bash
    npm install
    ```
3. Start the program:
    ```bash
    npm start
    ```
